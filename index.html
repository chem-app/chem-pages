プライバシーポリシー

Chem - BL創作漫画・イラストSNS

Chem Labs（以下「当方」といいます）は、「Chem - BL創作漫画・イラストSNS」（以下「本サービス」といいます）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

第1条（収集する情報）

当方は、本サービスの提供にあたり、以下の情報を収集します。

1. ユーザーが提供する情報

  • メールアドレス
  • パスワード
  • 表示名
  • ユーザー名
  • 生年月日
  • 自己紹介文（プロフィール）
  • プロフィール画像
  • 投稿コンテンツ（画像、キャプション、コメント、スタンプ）
  • カップリングの設定情報（お気に入り・ブロック）
  • コンテンツ表示設定（成人向けコンテンツの表示可否）
  • 通知設定

2. 自動的に収集する情報

  • デバイス識別子（インストールID）
  • デバイスのプラットフォーム（iOS / Android）
  • OSバージョン
  • プッシュ通知用トークン
  • アプリの利用状況（Firebase Analyticsによる）
  • アプリのクラッシュ情報（Firebase Crashlyticsによる）
  • CDNアクセスログ（IPアドレスを含み、90日間保持）

第2条（利用目的）

収集した情報は、以下の目的で利用します。

  1. アカウントの作成・認証・管理
  2. 本サービスの機能提供（タイムライン表示、検索、投稿、コメント、スタンプ、ブックマーク、フォロー等）
  3. プッシュ通知の送信
  4. コンテンツ表示のパーソナライズ（カップリング設定、フォロー関係、コンテンツ表示設定に基づく）
  5. アプリの安定性の監視・改善（クラッシュレポート分析）
  6. 利用状況の分析・サービス改善
  7. お問い合わせ対応
  8. 利用規約の遵守状況の確認・不正利用の防止
  9. 本ポリシーの変更等に関する通知

第3条（第三者サービスの利用）

当方は、本サービスの提供にあたり、以下の第三者サービスを利用しています。これらのサービスに対して、それぞれの利用目的に必要な範囲で情報が提供されます。

Supabase, Inc.（米国）

  • 提供する情報: メールアドレス、パスワード（ハッシュ化済み）、認証情報
  • 利用目的: ユーザー認証・アカウント管理
  • プライバシーポリシー: https://supabase.com/privacy

Google LLC（米国）— Firebase

  • 提供する情報: デバイス情報、プッシュ通知トークン、ユーザーID、アプリ利用状況、クラッシュ情報
  • 利用目的:
    • Firebase Cloud Messaging: プッシュ通知の送信
    • Firebase Analytics: 利用状況の分析
    • Firebase Crashlytics: アプリのクラッシュ検知・安定性向上
  • プライバシーポリシー: https://policies.google.com/privacy

Amazon Web Services, Inc.（米国）— AWS

  • 提供する情報: ユーザーがアップロードした画像、データベースに保存される情報
  • 利用目的: データベースホスティング、画像ストレージ（S3）、コンテンツ配信（CloudFront）
  • プライバシーポリシー: https://aws.amazon.com/privacy/

上記の第三者サービスの利用に伴い、ユーザーの情報が日本国外（米国）に移転される場合があります。各サービス提供者は、それぞれのプライバシーポリシーに基づき適切なデータ保護措置を講じています。

第4条（個人情報の第三者提供）

当方は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命、身体または財産の保護のために必要がある場合であって、ユーザーの同意を得ることが困難である場合
  4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、ユーザーの同意を得ることが困難である場合
  5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、ユーザーの同意を得ることにより当該事務の遂行に支障を及ぼすおそれがある場合
  6. 事業承継に伴い個人情報を提供する場合

第5条（情報の保存期間）

  1. アカウント情報および投稿コンテンツは、ユーザーが削除するかアカウントを退会するまで保持します。
  2. CDNアクセスログ（IPアドレスを含む）は、取得から90日間保持した後に削除します。
  3. Firebase Analytics および Crashlytics のデータは、各サービスの保持ポリシーに従います。

第6条（ユーザーの権利）

ユーザーは、以下の操作を本サービス内で行うことができます。

  1. プロフィールの編集・変更 — 表示名、自己紹介文、プロフィール画像、生年月日の変更
  2. 投稿の削除 — 自身の投稿、コメントの削除
  3. プッシュ通知の管理 — 通知種別ごとのオン・オフ切り替え、プッシュ通知の無効化
  4. コンテンツ表示設定の変更 — 成人向けコンテンツの表示・非表示の切り替え
  5. アカウントの退会（削除） — アカウントを削除することで、登録情報および関連するデータを削除できます

個人情報の開示、訂正、利用停止等のご請求については、第10条記載のお問い合わせ先までご連絡ください。

第7条（安全管理措置）

当方は、収集した個人情報の漏洩、滅失または毀損の防止のため、以下の安全管理措置を講じています。

  1. 通信の暗号化（HTTPS/TLS）
  2. パスワードのハッシュ化
  3. データベースおよびストレージの暗号化（保存時）
  4. アクセストークンによる認証（JWT）
  5. ストレージへの公開アクセスの遮断

第8条（Cookieの利用）

本サービスは、ユーザーのセッション管理にHTTP Cookieを使用しません。認証はJWTトークンベースで行われ、トークンはデバイス内のセキュアストレージに保存されます。

なお、第3条に定める第三者サービスが、それぞれの機能の提供に際して類似の技術を使用する場合があります。

第9条（本ポリシーの変更）

  1. 当方は、必要に応じて本ポリシーを変更することがあります。
  2. 変更後のプライバシーポリシーは、本サービス内またはウェブサイト上に掲示した時点から効力を生じるものとします。
  3. 個人情報の利用目的の変更等、ユーザーの同意が必要な変更については、同意を得た上で適用します。
