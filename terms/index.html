利用規約

Chem - BL創作漫画・イラストSNS

この利用規約（以下「本規約」といいます）は、Chem Labs（以下「当方」といいます）が提供する「Chem - BL創作漫画・イラストSNS」（以下「本サービス」といいます）の利用条件を定めるものです。ユーザーの皆さま（以下「ユーザー」といいます）は、本規約に同意の上、本サービスを利用するものとします。

第1条（適用）

  1. 本規約は、ユーザーと当方との間の本サービスの利用に関わる一切の関係に適用されるものとします。
  2. 当方が本サービス上で掲載するガイドライン、ヘルプ、注意事項等（以下「個別規定」といいます）は、本規約の一部を構成するものとします。
  3. 本規約と個別規定の内容が異なる場合は、個別規定が優先されるものとします。

第2条（定義）

本規約において、次の各号に掲げる用語の意味は、当該各号に定めるとおりとします。

  1. 「本サービス」とは、当方が提供するBL創作漫画・イラストの投稿、閲覧および交流を目的としたSNSサービスをいいます。
  2. 「ユーザー」とは、本サービスに登録し、または本サービスを利用するすべての個人をいいます。
  3. 「投稿コンテンツ」とは、ユーザーが本サービスに投稿、送信またはアップロードしたイラスト、漫画、画像、テキスト、コメント、タグ情報その他一切のコンテンツをいいます。
  4. 「知的財産権」とは、著作権、特許権、実用新案権、意匠権、商標権その他の知的財産権（それらの権利を取得し、またはそれらの権利につき登録等を出願する権利を含みます）をいいます。

第3条（アカウント登録）

  1. 本サービスの利用を希望する者は、本規約に同意の上、当方の定める方法によりアカウント登録を行うものとします。
  2. 当方は、登録の申請者が以下のいずれかに該当すると判断した場合、登録を拒否することがあります。
    1. 登録情報に虚偽の内容が含まれている場合
    2. 過去に本規約に違反したことがある場合
    3. その他、当方が登録を適当でないと判断した場合

第4条（アカウント管理）

  1. ユーザーは、自己の責任において、本サービスのアカウント情報（メールアドレス、パスワード等）を適切に管理するものとします。
  2. ユーザーは、アカウントを第三者に譲渡、貸与、または第三者と共用することはできません。
  3. 登録されたアカウント情報を用いてなされた一切の行為は、当該アカウントを保有するユーザー本人の行為とみなします。
  4. アカウント情報の管理不十分、第三者の使用等によって生じた損害に関する責任はユーザーが負うものとし、当方の故意または重大な過失がある場合を除き、当方は一切の責任を負いません。

第5条（投稿コンテンツと知的財産権）

  1. 投稿コンテンツに関する著作権その他の知的財産権は、当該投稿コンテンツを創作したユーザーまたは当該権利を有する第三者に帰属します。本サービスへの投稿により、当方に権利が移転することはありません。
  2. ユーザーは当方に対し、本サービスの提供、運営、維持、改善および広報のために必要な範囲において、投稿コンテンツを非独占的かつ無償で利用する権利（複製、翻案、編集、変換、公衆送信、表示、配信等を含む）を許諾するものとします。
  3. ユーザーは、当方による本サービスの提供・運営に必要な範囲での利用に関して、著作者人格権を行使しないものとします。
  4. 何人も、本サービス上の投稿コンテンツを、機械学習、人工知能（AI）の学習、生成AIのトレーニングその他これに類する目的のために、当該権利者の許可なく取得、収集または利用してはなりません。

第6条（二次創作・ファン活動に関する特則）

  1. 本サービスでは、既存の著作物を題材とした二次創作コンテンツ（ファンアート、ファンコミック等）の投稿が行われることがあります。
  2. ユーザーは、二次創作コンテンツを投稿する場合、原著作物の著作権者の権利を侵害しないよう十分に注意し、自己の責任において投稿するものとします。
  3. 当方は、二次創作コンテンツの投稿を許容するプラットフォームを提供する立場にありますが、各投稿コンテンツが第三者の権利を侵害していないことを保証するものではありません。
  4. 権利者またはその正当な代理人から、著作権侵害等を理由とする削除要請があった場合、当方は当該コンテンツを削除する等の適切な措置を講じます。
  5. ユーザーは、二次創作コンテンツの投稿に関して第三者との間で紛争が生じた場合、自己の費用と責任においてこれを解決するものとし、当方に損害を与えないものとします。

第7条（禁止事項）

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはなりません。

  1. 法令または公序良俗に違反する行為
  2. 犯罪行為または犯罪を助長する行為
  3. 第三者の著作権、商標権、肖像権、プライバシー権、名誉その他の権利または利益を侵害する行為
  4. 児童ポルノまたは児童虐待に該当する、またはそのおそれのあるコンテンツを投稿する行為

  6. 過度に暴力的、残虐的、差別的または憎悪を助長する表現を含むコンテンツを投稿する行為
  7. 本サービス、他のユーザーまたは第三者のサーバーやネットワークの機能を破壊または妨害する行為
  8. 本サービスの運営を妨害するおそれのある行為
  9. 不正アクセスまたはこれを試みる行為
  10. 他のユーザーの個人情報または利用情報を不正に収集する行為
  11. 他のユーザーに対する誹謗中傷、嫌がらせ、脅迫、ストーキング等の行為
  12. 反社会的勢力への利益供与または関与する行為
  13. 他のユーザーまたは第三者になりすます行為

  19. その他、当方が不適切と判断する行為

第8条（成人向けコンテンツ）

  1. ユーザーは、性的表現を含むコンテンツを投稿する場合、必ず成人向け（R-18）の設定を行った上で投稿するものとします。
  2. 成人向けコンテンツの閲覧は、当方が定める年齢要件を満たすユーザーに限定されます。
  3. 当方は、成人向け設定が適切に行われていないと判断したコンテンツに対し、設定の変更または削除等の措置を講じることがあります。

第9条（実在人物を題材とするコンテンツ）

  1. ユーザーは、実在する人物を題材としたコンテンツを投稿する場合、当該人物の名誉、プライバシー、肖像権その他の権利または利益を侵害しないよう十分配慮するものとします。
  2. 当方は、権利侵害または社会的影響のおそれがあると判断した場合、当該コンテンツを削除することがあります。

第10条（コンテンツの削除・利用制限）

  1. 当方は、投稿コンテンツが以下のいずれかに該当する、または該当するおそれがあると判断した場合、事前の通知なく当該コンテンツを削除し、またはユーザーのアカウントを停止・削除できるものとします。

    2. 法令に違反する場合
    3. 第三者の権利を侵害する場合
    4. 権利者またはその正当な代理人から削除の要請があった場合
    5. その他、当方が本サービスの運営上必要と判断した場合
  2. 当方は、前項の措置によりユーザーに生じた損害について、当方の故意または重大な過失がある場合を除き、一切の責任を負いません。

    2. 地震、落雷、火災、停電、天災等の不可抗力により本サービスの提供が困難となった場合

第13条（退会）
